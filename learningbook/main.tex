\subsection{Переменные}
 \fontsize{14pt}{14pt}\selectfont
Переменные в Zig объявляются с помощью ключевого слова \texttt{var} или \texttt{const}. Например:

\begin{lstlisting}
const x = 5; // Константа
var y = 10;   // Переменная
\end{lstlisting}
